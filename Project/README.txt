Tartan Adventure is a 2D side-scroller game where the player's objective is to avoid enemies and obstacles while progressing through the level.


To play Tartan Adventure you will need to have the pygame module installed. Instructions for how to do so can be found at: 
https://cs.hofstra.edu/docs/pages/guides/InstallingPygame.html 

Note: To play the multiplayer mode you will need a valid account (Andrew ID) registered on the server (86.36.46.10) 

To play the single player mode run SinglePlayer.py

To play the online multiplayer mode run Multiplayer.py. 


(Separate single player and multiplayer files were created due to Tkinter's incompatibility with pygame)
